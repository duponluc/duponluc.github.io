]]> http://www.duponluc.be/particuliere-woningen/feed/ 0 Verwarming http://www.duponluc.be/verwarming/ http://www.duponluc.be/verwarming/#comments Wed, 25 Aug 2010 14:20:26 +0000 admin http://127.0.0.1/wordpress/?p=138 Wij zijn reeds vele jaren actief in elektrische verwarming. Dit soort verwarming is bekend geworden als een nacht opslag verwarming via zijn verwarmingstoestellen, ook gekend als accumulatoren. Deze toestellen laden stroom via het net op om het dan geleidelijk terug af te geven. Het laden gebeurt typisch wannneer het financieel interessant is (nachttarief). Naast deze accumulatoren leggen wij eveneens vloerverwarming.

Sinds de invoering van de EPB normering, wordt elektrische verwarming minder positief beoordeeld. Een goed alternatief is verwarming via airconditioning met warmtepompen.
